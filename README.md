# 🏨 Puneeth Grand Hotel - Website Project

A simple, responsive hotel website built using **HTML5, CSS3, and JavaScript**.  
This project is created for the MANTRA 2026 Frontend Assignment.

---

---

## ✨ Features

- Responsive design for mobile, tablet, and desktop
- Multi-page website (Home, About, Rooms, Gallery, Booking, Contact)
- Modern UI with clean layout
- Image gallery with hotel facilities
- Booking form with JavaScript validation
- Contact form with validation
- SEO-friendly structure (title, meta tags, headings, alt text)

---

## 🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox + Media Queries)
- JavaScript (Form validation)
- GitHub (Code hosting)
- Netlify (Deployment)

---

## 📄 Pages Overview

### 🏠 Home Page
- Hero section

### ℹ️ About Page

### 🛏️ Rooms Page
- Room types with prices
- Booking buttons

### 🖼️ Gallery Page

- JavaScript validation

### 📞 Contact Page
- Contact details
- Contact form

## 🚀 How to Run Project

1. Download or clone the repository
2. Open folder in VS Code
3. Open `index.html` in browser
4. Navigate through pages

---

## 📦 Deployment (Netlify)

1. Go to https://www.netlify.com/
2. Drag and drop the project folder
3. Get live link
4. Add link in README
